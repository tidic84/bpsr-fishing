1. exclamation_point.png

2. button_continue.png
   ├─ Quoi : Le bouton "Continue" dans le menu de fin
   ├─ Quand : Après avoir attrapé un poisson avec succès
   └─ Usage : Le bot clique dessus pour fermer le menu et recommencer

3. fishing_ready.png
   ├─ Quoi : L'indicateur qui montre que la canne est prête
   ├─ Quand : Après avoir terminé une pêche (succès ou échec)
   └─ Usage : Le bot attend que cet indicateur apparaisse avant de recommencer

4. use_button.png
   ├─ Quoi : Le bouton "Use" dans le menu de sélection des cannes
   ├─ Quand : Menu ouvert avec la touche "," (virgule)
   └─ Usage : Le bot clique dessus pour équiper une nouvelle canne

5. rod_broken.png
   ├─ Quoi : Un indicateur visuel qui apparaît quand la canne est cassée
   ├─ Quand : Quand la canne se casse
   └─ Usage : Permet de détecter si la canne est cassée sans ouvrir le menu
