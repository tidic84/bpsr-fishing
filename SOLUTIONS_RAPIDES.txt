Si l'étape 1 (clic de démarrage) ne fonctionne pas, voici les solutions les plus
courantes par ordre de probabilité :

┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION #1 : MODE D'AFFICHAGE DU JEU (90% des cas)                       │
└──────────────────────────────────────────────────────────────────────────────┘

Le problème le plus fréquent !

1. Ouvrez les PARAMÈTRES de votre jeu
2. Cherchez "Mode d'affichage" / "Display Mode"
3. Changez en:
   
   ✓ FENÊTRÉ SANS BORDURE (Borderless Windowed)
   ou
   ✓ FENÊTRÉ (Windowed)
   
   PAS:
   ✗ PLEIN ÉCRAN (Fullscreen)

4. Redémarrez le bot

Explication: Le plein écran exclusif empêche PyAutoGUI de contrôler le jeu.

┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION #2 : DÉSACTIVER LES OVERLAYS (5% des cas)                        │
└──────────────────────────────────────────────────────────────────────────────┘

Si Solution #1 ne marche pas, désactivez les overlays :

→ DISCORD : Paramètres → Overlay de jeu → Désactiver
→ STEAM   : Paramètres → En jeu → Désactiver overlay
→ NVIDIA  : GeForce Experience → Paramètres → Désactiver overlay
→ XBOX    : Windows → Paramètres → Jeux → Désactiver Game Bar

┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION #3 : TESTER LA POSITION (3% des cas)                             │
└──────────────────────────────────────────────────────────────────────────────┘

Peut-être que la position de clic est incorrecte :

1. Lancez le test :
   > python test_clic_peche.py

2. Suivez les instructions pour vérifier la position

3. Si nécessaire, recalibrez :
   > python calibration.py

┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION #4 : PERMISSIONS (1% des cas)                                     │
└──────────────────────────────────────────────────────────────────────────────┘

Si rien d'autre ne fonctionne :

1. Lancez PowerShell en ADMINISTRATEUR
2. Naviguez vers le dossier :
   > cd C:\chemin\vers\automation_peche
3. Activez l'environnement :
   > venv\Scripts\activate
4. Relancez le bot :
   > python fishing_bot.py

┌──────────────────────────────────────────────────────────────────────────────┐
│ ✅ SOLUTION #5 : ANTIVIRUS (1% des cas)                                       │
└──────────────────────────────────────────────────────────────────────────────┘

Votre antivirus peut bloquer PyAutoGUI :

1. Ouvrez votre antivirus
2. Ajoutez une EXCEPTION pour :
   - Le dossier automation_peche
   - python.exe
