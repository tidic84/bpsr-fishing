┌──────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 1 : INSTALLATION                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

1. Ouvrez un terminal dans ce dossier

2. Créez un environnement virtuel :
   > python -m venv venv

3. Activez l'environnement virtuel :
   > venv\Scripts\activate          (Windows)
   > source venv/bin/activate       (Linux/macOS)

4. Installez les dépendances :
   > pip install -r requirements.txt

┌──────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 2 : CALIBRATION (PREMIÈRE UTILISATION)                                 │
└──────────────────────────────────────────────────────────────────────────────┘

1. Lancez le jeu et positionnez-vous à l'endroit de pêche

2. Lancez la calibration :
   > python calibration.py

3. Suivez les instructions pour capturer :
   - Le point d'exclamation (!)
   - Le bouton "Continue"
   - La position de clic pour démarrer

┌──────────────────────────────────────────────────────────────────────────────┐
│ ÉTAPE 3 : LANCER LE BOT                                                      │
└──────────────────────────────────────────────────────────────────────────────┘

1. Lancez le bot :
   > python fishing_bot.py

   OU double-cliquez sur :
   > start_bot.bat    (Windows uniquement)

2. Choisissez le mode de confiance (1 = recommandé)

3. Choisissez le nombre de cycles (vide = illimité)

4. Mettez le jeu au premier plan

5. Le bot démarre après 5 secondes !

┌──────────────────────────────────────────────────────────────────────────────┐
│ CONTRÔLES PENDANT L'EXÉCUTION                                                 │
└──────────────────────────────────────────────────────────────────────────────┘

  ESC      → Arrêt d'urgence
  Ctrl+C   → Arrêt dans le terminal

┌──────────────────────────────────────────────────────────────────────────────┐
│ PROBLÈMES COURANTS                                                            │
└──────────────────────────────────────────────────────────────────────────────┘

❌ "Point d'exclamation non détecté"
   → Recalibrez avec une capture plus précise
   → Essayez le mode souple (option 3)

❌ "Bouton Continue non détecté"
   → Augmentez QTE_WAIT_TIME dans config_exemple.py
   → Recalibrez le bouton Continue

❌ "Erreur lors de l'import de cv2"
   → Réinstallez : pip install --force-reinstall opencv-python

❌ Le bot clique au mauvais endroit
   → Recalibrez les positions
   → Assurez-vous que le jeu est à la même position/résolution

┌──────────────────────────────────────────────────────────────────────────────┐
│ CONSEILS IMPORTANTS                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

  ✓ Gardez toujours la même résolution de jeu
  ✓ Ne déplacez pas la fenêtre du jeu entre les sessions
  ✓ Surveillez le bot, surtout au début
  ✓ Recalibrez si vous changez les paramètres graphiques du jeu
